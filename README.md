# PayShield 🛡️

PayShield is a real-time UPI fraud detection Android application that protects users **before they enter their PIN**.

## 🚨 Problem
Users blindly trust QR codes and UPI IDs, leading to fraud.

## ✅ Solution
PayShield uses Android Accessibility Service to monitor payment screens and shows a warning overlay before confirmation.

## ⚙️ Features
- Detects payment screen
- Extracts UPI ID, name, amount
- Calculates risk score
- Firebase fraud report checking
- Real-time warning overlay

## 🧠 Tech Stack
- Java
- XML
- Android Accessibility Service
- Firebase Realtime Database

## 🔄 Workflow
1. Detect payment screen
2. Extract data
3. Calculate risk
4. Show warning overlay

## 🚀 Novelty
Prevents fraud BEFORE payment instead of after.
